ChatGPT said:

To attach a real-time map and a share button in your div, you can use an embedding solution from a mapping service like Google Maps. Here's a simple implementation using the Google Maps API, along with a share button.

Step 1: Embed Google Maps

First, you need to embed the Google Maps iframe in your div. You can also use the Google Maps JavaScript API for more interactive features.

Step 2: Create a Share Button

You can create a button that allows users to share the location via social media or messaging apps.

Sample Code

Here's how you can do this:

html
<div class="service-item position-relative">
    <div class="icon"><i class="bi bi-camera-video-fill" style="color: green;"></i></div>
    <h4><a href="" class="stretched-link"></a></h4>
    <p>Utilize AI-powered image processing to identify and categorize waste, ensuring timely collection and management.</p>

    <!-- Real-time Map -->
    <div id="map" style="width: 100%; height: 300px;"></div>

    <!-- Share Button -->
    <button onclick="shareLocation()" style="margin-top: 10px;">Share Location</button>
</div>

<!-- Include Google Maps JavaScript API -->
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" async defer></script>

<script>
    function initMap() {
        // Set the location
        const location = { lat: -34.397, lng: 150.644 }; // Replace with your desired coordinates

        // Create a map
        const map = new google.maps.Map(document.getElementById("map"), {
            zoom: 15,
            center: location,
        });

        // Add a marker
        const marker = new google.maps.Marker({
            position: location,
            map: map,
        });
    }

    function shareLocation() {
        const locationUrl = "https://maps.google.com/?q=" + encodeURIComponent("-34.397,150.644"); // Replace with the current coordinates
        const shareText = "Check out my location!";
        const shareLink = `${shareText} ${locationUrl}`;

        // Using the Web Share API if available
        if (navigator.share) {
            navigator.share({
                title: 'Location Share',
                text: shareText,
                url: locationUrl,
            })
            .then(() => console.log('Location shared successfully'))
            .catch((error) => console.log('Error sharing location', error));
        } else {
            // Fallback for browsers that do not support the Web Share API
            alert('Share this link: ' + shareLink);
        }
    }
</script>

Explanation:

  1. Map: The <div id="map"> creates a space for the Google Map. You must replace YOUR_API_KEY with your actual Google Maps API key.
  2. JavaScript Functions:
    • initMap(): Initializes the map and sets the location (change coordinates as needed).
    • shareLocation(): Creates a shareable link to the location. If the Web Share API is supported by the browser, it prompts the user to share; otherwise, it shows an alert with the shareable link.
  3. Button: The button calls the shareLocation() function when clicked.

Note:

Make sure to enable the necessary APIs on the Google Cloud Console to use the Google Maps JavaScript API, and adjust the CSS styles as needed to fit your design.
